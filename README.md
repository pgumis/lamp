# LAMP

Każdy folder posiada plik Dockerfile + plik konfiguracyjny

## Użycie

```bash
docker-compose pull

docker-compose up -d
```

## Opis
docker-compose definiuje aplikacje z trzema kontenerami: Apache, Mysql, PHP
